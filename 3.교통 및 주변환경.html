한신대학교

  • 교통 및 주변 환경

경기 남부 최대 상권으로 개발되는 1호선 병점(한신대)역 복합타운 부근이며, GTX C노선 연결과 동탄트램도 들어올 예정으로 교통 요충지에 위치하고 있다.

  • 전철
    1호선 병점(한신대)역 하차, 2번출구 버스정류장에서 마을버스(55, 56) 이용

  • 버스
    8번버스 이용: 한신대후문 하차후 도보이동

    1) 수원에서 오는 길: 망포역 -> 병점사거리 -> 한일타운 -> 한신대후문
    2) 오산에서 오는 길: 오산역 -> 세마역 -> 한신대후문
    3) 한신대후문 하차 -> 송암관까지 도보 5~7분소요

    7번버스 이용 : 효성아파트후문 하차후 도보이동 1) 병점에서 오는 길: 롯데시네마 -> 병점사거리 -> 효성아파트후문 2) 오산에서 오는 길: 오산역 -> 세마역 -> 효성아파트후문 3) 효성아파트후문 하차 -> 송암관까지 도보 5~7분소요
